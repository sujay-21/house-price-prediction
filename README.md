# House Price Prediction Web App

A machine learning web application that predicts house prices using the California Housing dataset.  
The model is trained using Linear Regression and deployed using Flask.

---

## Features
- Predict house prices based on housing features
- Machine learning model trained using Scikit-learn
- Interactive web interface using Flask
- Real-time prediction from user input

---

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML

---

## Dataset
This project uses the **California Housing dataset** provided by Scikit-learn.

Features used for prediction:

- MedInc (Median Income)
- HouseAge
- AveRooms
- AveBedrms
- Population
- AveOccup
- Latitude
- Longitude
